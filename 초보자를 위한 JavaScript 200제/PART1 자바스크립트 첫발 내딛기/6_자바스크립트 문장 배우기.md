표현식 명령문
```
functions greeting() {
  "hello" // 명령문 대신 표현식 사용 가능
  "Chloe" + 3 // 명령문 대신 표현식 사용 가능
  greeting()
}

greeting(if(true) { console.log("It is not acceptable") })
// 표현식 대신 명령문 사용 불가능
```
